# bingo-generator
Simple bingo generator.

<b>Dependencies </b>(ubuntu):
<br>
<code>
	build-essential libsfml-dev
</code>
<br>
	
<b>Usage</b>
<br>
-s [phrases source]
-f [font source]
-b [bonus image source"
-i [quantity]

<b>Build</b>
<br>
<code>
	git clone https://github.com/hjaremko/bingo-generator.git && cd bingo-generator 
</code>
<br>
<code>
	make
</code>
<br>
