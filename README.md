# bingo-generator
Simple bingo generator.

<b>Dependencies </b>(ubuntu):
<br>
<code>
	build-essential libsfml-dev
</code>
<br>
	
<b>Usage</b>
<br>
-s "[phrases source]"
-i [quantity]
-b "[bonus image source"]

<b>Build</b>
<br>
<code>
	git clone https://github.com/Freelan/bingo-generator.git && cd bingo-generator 
</code>
<br>
<code>
	make
</code>
<br>
